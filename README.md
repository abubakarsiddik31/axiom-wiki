# Axiom Wiki

**The wiki that maintains itself.**

An open-source, CLI-first tool that turns a folder of raw documents into a persistent, AI-maintained personal knowledge base.

> Knowledge that compounds.

## Status

Work in progress — V1 under active development.

## Stack

TypeScript · Mastra · Gemini API · MCP · Ink

## Install

```bash
npm install -g axiom-wiki
# or
npx axiom-wiki init
```

## Usage

```bash
axiom-wiki init        # First-time setup wizard
axiom-wiki ingest      # Ingest files from raw/ folder
axiom-wiki query       # Interactive chat against your wiki
axiom-wiki status      # Wiki statistics
axiom-wiki mcp         # Start MCP server for Claude Code / Cursor
```
