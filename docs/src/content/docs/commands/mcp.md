---
title: mcp
description: Start MCP server for Claude Code / Cursor integration.
---

```bash
axiom-wiki mcp
```

Starts the MCP server via stdio transport. This is used by Claude Code, Cursor, and other MCP-compatible clients to interact with your wiki programmatically.

See the [MCP Integration guide](/axiom-wiki/guides/mcp/) for setup instructions and the full list of available tools.
